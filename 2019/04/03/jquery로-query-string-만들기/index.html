querystring jquery 로 쉽고 명확하게 만들기

javascript상에서 쿼리스트링을 생성하여 redirect하거나, 비동기로 요청을 해야하는 경우가 있다.
간단하게는 string을 직접 생성하여 사용하는 방법도 있다.
